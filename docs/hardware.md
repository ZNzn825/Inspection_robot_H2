## 接线说明

### 底盘与工控机
- 底盘 CAN 接口通过 USB-CAN 适配器连接工控机 USB 口
- CAN 协议：标准帧 (ZCAN_20_MSG)，8 字节数据区

### 机械臂与工控机
- 工控机 → 交换机 → 机械臂控制器
- 控制器 IP：`192.168.1.10`，端口：`8899`
- SDK 连接后需先登录再上电

### 传感器接线
- 氢气传感器：RS485 → USB-TTL 转换器 → ttyS4，9600bps，Modbus RTU
- 火焰传感器：RS485 → USB-TTL 转换器 → ttyS0，9600bps

### 激光雷达
- Livox MID360 通过网线直连工控机
- 使用 `livox_ros_driver2` 驱动

### ZED 2 相机
- USB 3.0 直连工控机
- 需要 ZED SDK（CUDA 加速可选）

## 网络配置

```
工控机  ←→ 交换机 ←→ 机械臂控制器 (192.168.1.10)
                    ←→ Livox MID360   (192.168.1.50)
                    ←→ UTi586A 红外    (根据实际配置)
```
